Tentang Sistem

psychology Cara Kerja Algoritma SVM

Support Vector Machine (SVM) adalah model pembelajaran terawasi (supervised learning) yang mencari batas pemisah optimal (hyperplane) dengan margin terbesar antara kelas pelanggan yang "Churn" dan "Tetap Berlangganan".

Visualisasi Pembagian Kelas Hyperplane

Ilustrasi 2D: Support vector (lingkaran berbingkai) menentukan posisi garis pemisah optimal (biru).

Berbeda dengan model probabilitas biasa, SVM berfokus pada titik data tersulit yang terletak paling dekat dengan keputusan (Support Vectors). Dengan menyeimbangkan bias melalui standarisasi fitur, model SVM mencapai akurasi tinggi bahkan ketika pola churn pelanggan bersifat non-linear.

science Metodologi Pengembangan

1

Eksplorasi dan pembersihan dataset Telco Customer Churn, termasuk penanganan missing values dan encoding fitur kategorikal.

2

Normalisasi fitur menggunakan StandardScaler untuk menyeimbangkan skala data sebelum pelatihan model SVM.

3

Pelatihan model SVM dengan kernel RBF, optimasi hyperparameter, dan evaluasi menggunakan K-Fold Cross Validation.

4

Deployment model ke web dashboard interaktif menggunakan Flask REST API dan antarmuka responsif berbasis JavaScript.

Sistem ini telah melalui proses pengujian dan validasi prediktif. Data uji coba (20 pelanggan dummy) tersedia dalam file CSV di folder proyek.
